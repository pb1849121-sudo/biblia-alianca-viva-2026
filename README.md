# Aliança Viva

## Uma tradução bíblica modernizada e atual

Bem-vindo ao projeto **Aliança Viva**, uma tradução das Escrituras em linguagem modernizada, atual e acessível, desenvolvida para transmitir a mensagem bíblica com clareza aos leitores de hoje.

## Sobre o projeto

A **Aliança Viva** busca unir fidelidade ao texto bíblico e uma comunicação compreensível para o mundo contemporâneo.

O projeto utiliza o formato **USFM (Unified Standard Format Markers)**, padrão utilizado em projetos de tradução e publicação bíblica.

## Estrutura

## Formatos planejados

A partir dos arquivos USFM, o projeto poderá ser disponibilizado em:

- EPUB
- HTML
- PDF
- Aplicativos bíblicos compatíveis

## Objetivo

Disponibilizar uma tradução bíblica moderna, organizada e acessível para leitores digitais, permitindo que a mensagem das Escrituras alcance pessoas em diferentes plataformas.
